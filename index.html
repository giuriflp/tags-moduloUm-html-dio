O que e quem são os Clientes

A palavra Clientes representa os usuários que acessam o website atraves de um browser.

O que e quem são os servidores

A palavra Servidor representa o computador que irá fazer todo o trabalho de receber requisições e responde-las para que os clientes possam acessar o site, fazer alguma busca, etc.

Estrutura básica de um HTML

A estrutura básica de um arquivo html sempre tem que conter as tags HTML, head (dentro da tag head, o title) e a tag body.

Diferenças de listas ordenadas e não ordenadas:

As listas não ordenadas são listas que não importa a ordem dos elementos, como uma bullet list. Ex:

  • Item 1
  • Item 2

Já as listas ordenadas, os elementos tem uma ordem. Ex:

  1. Item um
  2. Item dois
